Pacman Game

How to Run the Source Code

1. Install Python:
   - Ensure you have Python 3.10 or higher installed on your system. You can download it from https://www.python.org/.

2. Install Required Packages:
   - Open a terminal in the `Source` directory.
   - Run the following command to install the required dependencies:
        pip install -r requirements.txt

3. Run the Game:
   - Execute the `main.py` file using the following command:
        py main.py

4. Select a Level:
   - After running the game, a window will appear.
   - Press keys 1 to 6 to select the corresponding level.

## Notes
- Ensure all required assets (e.g., images, sounds) are present in the images folder.
- If you encounter any issues, ensure your working directory is set to Source.
- For best performance, run the game in a Python virtual environment.
